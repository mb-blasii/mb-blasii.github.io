Duomondo

Mobile Game · Location-Based Gameplay · Backend

Project Overview

Duomondo is a location-based mobile game inspired by PokemonGO exploration mechanics, designed to encourage players to discover places and interact with their surroundings.


The project combines GPS-based gameplay with progression systems, collectible content, quests and light social mechanics, tailored for casual and young audiences.


The game was released exclusively in Italy. I worked on the project as a Unity Developer, focusing on gameplay systems, mobile optimization, technical implementation and backend design.

Features

  • Location-Based Gameplay Real-world GPS positioning used to drive exploration and in-game interactions.
  • Google Geospatial Integration Positioning and map data integrated with Geospatial and Google Maps API for accurate location placement.
  • Progression & Collection Systems Players unlock content, rewards, and achievements by visiting locations and completing activities.
  • Mobile-Oriented UX Designed specifically for touch input, short play sessions, and outdoor usage.
  • Map Integration Interactive map interface displaying nearby points of interest and gameplay elements.
  • Live Data & Content Updates Backend-driven content allowing new locations and events without app updates.
  • Performance Optimization Careful handling of GPS polling, battery usage, and mobile performance constraints.
