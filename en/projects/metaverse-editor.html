Metaverse World Editor

Runtime Editing · Modular UI · Data-Driven Worlds

Project Overview

A custom runtime world editor designed for a metaverse application, allowing users to create, modify, and manage virtual environments directly in-game.

The editor interface mimics the Unity user interface to provide a familiar experience to more experienced developers, while removing all the unnecessary complexity for artists and casual world builders.


Built with a modular architecture, the editor supports dynamic entity management, component-based systems, and real-time world updates without requiring traditional development tools.


The software focuses on scalability, performance, and usability, enabling both creators and non-technical users to shape interactive worlds through a clean, intuitive interface.

Features

  • Runtime World Editing Edit scenes, entities, and properties on a runtime application without needs to learn to use a full game engine.
  • Entity & Component Architecture Custom MetaEntity and MetaComponent system for flexible, data-driven world building.
  • Dynamic Asset Browser Runtime-loaded assets with extensible filtering, previews, and category management. Synced with the backend server to retrieve all user and store assets.
  • Live Scene Reconstruction Worlds can be rebuilt from serialized data to support persistence and rapid iteration. Data is optimized to achieve minimum network usage and fast loading times without information loss.
  • Performance-Oriented Design Optimized runtime workflows to keep editing responsive even in complex environments.
  • Editor & Runtime Separation Core systems shared between the editor and the client to ensure consistency and maintainability.
