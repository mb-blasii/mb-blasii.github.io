Animator Dialogue System

Animation Tools · Narrative Systems · Unity Editor Extension

Project Overview

Unity tool designed as an easy alternative to my Node-Based dialogue system to tightly integrate dialogue flows with unity animator system.


The system uses Animator states as the graphical source for dialogue nodes with StateMachineBehaviours providing all the logic needed to develop a strong dialogue flow.

Screenshots

Features

  • Visual State Machine Designer-friendly interface to link dialogue logic with animation flows.
  • Loops & Branches The states can be connected to form loops or have multiple out connections to branch the dialogue based on player choices.
  • Voiceover Each dialogue line can be voiced over. Voiceover also supports localization.
  • Localized Dialogue Dialogue can be localized using a string key to match the game's localization system.
  • Modular Architecture Built with extensibility in mind to support new animation systems and narrative tools.
