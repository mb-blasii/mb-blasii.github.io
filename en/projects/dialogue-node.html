Custom Node-Based Dialogue System

Dialogue Tools · Gameplay Systems · Unity Editor Extension

Project Overview

A custom node-based dialogue system built in Unity to design complex narrative flows through a visual, designer-friendly interface.

The system allows writers and designers to create branching conversations, conditional logic, and reusable dialogue components without touching code.

Developed entirely by me, covering architecture, editor tools, runtime systems, and data serialization.

Screenshots

Features

  • Player & NPC Dialogue Lines Full support for both player-driven and NPC-driven conversations.
  • Multi-Character Conversations Overridable nodes allow the change of speaker during a conversation.
  • Trigger Nodes Invoke events using string keys, allowing for animations and camera changes, calling actions and quest systems.
  • Callback Nodes Seamless integration with global systems through method and event callbacks.
  • Data-Driven Architecture Built around ScriptableObjects for modularity, scalability, and team-friendly workflows.
  • RunTime Optimization Dialogue graphs are optimized during the build process to minimize runtime overhead.
  • Branching & Looping Native support for complex narrative structures with conditional paths and persistent loops.
