Namazu's Fault

Award-Winning Game · Gameplay Programming · System Design

Project Overview

Namazu's Fault is a puzzle game developed during the Scientific Game Jam Tokyo 2025, where it received both the Golden Pixel Award and the Best Gameplay Award.


The project was created by a multidisciplinary team under tight time constraints. My main contributions focused on gameplay programming, system architecture, and technical implementation.

I worked on core mechanics, interaction logic, and overall game flow, ensuring that the gameplay systems and design balance were tightly integrated into a cohesive experience.
